musical playing cards

an interactive music-making experience

This program allows one to place different cards within frame to trigger different musical beats. Each card corresponds to a different beat, so by layering and timing the placement of the cards one can combine these beats and create music.

Programmed using Google’s Teachable Machine and p5.js
